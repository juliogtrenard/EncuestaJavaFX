# Titulo del proyecto

EncuestaJavaFX

## Descripción

Aplicación con JavaFX que recopila datos de una persona en una encuesta como su profesión, número de hermanos, aficiones...

## Primeros pasos

### Cómo usar

* Clona el repositorio en algún IDE y ejecuta el programa.
